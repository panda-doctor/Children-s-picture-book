# 儿童故事绘本应用开发计划

## 项目概述

开发一款面向儿童群体的故事绘本应用，支持上传/录入儿童故事文本，自动分章节并调用 Agnes Image 2.1 Flash 模型生成高清适配插画，最终排版生成交互式电子绘本。

## 技术架构

- **前端**: HTML5 + CSS3 + JavaScript（单页应用，无需框架，轻量易用）
- **后端**: Python Flask（提供 API 接口）
- **图像处理**: Pillow（用于排版生成 PDF）
- **图片生成**: Agnes Image 2.1 Flash API
- **存储**: 本地文件系统
- **目标年龄**: 3-8 岁儿童

## 功能模块

### 1. 故事录入模块
- 支持手动输入故事标题、作者、故事正文
- 支持上传 TXT/JSON 格式的故事文件
- 自动分章节解析（按段落或标记符分割）

### 2. 内容安全审核模块
- 敏感词过滤（儿童不当内容）
- 关键词白名单机制
- 违规内容提示与拦截

### 3. 插画生成模块
- 基于章节内容自动生成英文 Prompt
- 调用 Agnes Image 2.1 Flash API 生成插画
- 插画风格统一控制（卡通/水彩/蜡笔等儿童友好风格）
- 插画尺寸适配（1024x768 横版）

### 4. 绘本排版模块
- 图文混排布局
- 大字体、圆角边框等儿童友好设计
- 导出为 PDF 格式
- 支持在线翻页交互阅读

### 5. 交互阅读模块
- 翻页动画效果
- 语音朗读按钮（可选，后期扩展）
- 大按钮、简单导航（适配儿童操作）
- 色彩丰富、界面可爱

## 实施步骤

### 阶段一：基础框架搭建
1. 创建项目目录结构
2. 编写 `config.py` 配置文件（API Key、存储路径等）
3. 编写 `requirements.txt` 依赖文件
4. 编写 `app.py` Flask 主应用框架

### 阶段二：内容安全模块
1. 编写 `utils/content_filter.py`
   - 加载中英文敏感词库
   - 实现敏感词检测函数
   - 实现儿童内容安全评分
2. 创建敏感词库文件 `data/filters/chinese_words.txt` 和 `english_words.txt`

### 阶段三：故事解析模块
1. 编写 `utils/story_parser.py`
   - 支持 TXT 文件上传解析
   - 支持 JSON 格式故事解析
   - 自动分章节功能
   - 提取章节标题和正文

### 阶段四：插画生成模块
1. 编写 `utils/image_generator.py`
   - 封装 Agnes Image 2.1 Flash API 调用
   - 实现 Prompt 自动生成（将中文章节转为适合绘图的英文描述）
   - 实现图片下载和本地缓存
   - 支持多种绘画风格选择

### 阶段五：前端页面开发
1. 编写 `templates/index.html` - 主页（故事录入表单）
2. 编写 `static/css/style.css` - 全局样式（儿童友好配色、大按钮）
3. 编写 `static/js/app.js` - 主页面交互逻辑
4. 编写 `templates/editor.html` - 故事编辑器
5. 编写 `static/js/editor.js` - 编辑器逻辑
6. 编写 `templates/reader.html` - 绘本阅读器
7. 编写 `static/css/reader.css` - 阅读器样式
8. 编写 `static/js/reader.js` - 阅读器交互逻辑（翻页动画等）

### 阶段六：排版引擎
1. 编写 `utils/layout_engine.py`
   - 使用 Pillow 绘制绘本页面
   - 图文混排布局算法
   - 导出 PDF 功能
   - 儿童友好排版（大字体、圆角、彩色边框）

### 阶段七：API 路由整合
1. 在 `app.py` 中实现以下 API：
   - `POST /api/story/upload` - 上传故事
   - `POST /api/story/validate` - 内容安全校验
   - `POST /api/book/generate` - 生成绘本（分章节生成插画+排版）
   - `GET /api/books/list` - 获取绘本列表
   - `GET /api/books/<id>/read` - 在线阅读绘本
   - `GET /api/books/<id>/download` - 下载 PDF

### 阶段八：测试与优化
1. 单元测试各功能模块
2. 集成测试完整流程
3. 内容安全测试
4. 性能优化
5. 用户体验优化

## 核心设计要点

### 儿童友好 UI 设计
- 主色调：暖色系（橙色、黄色、天蓝）
- 字体：大号圆润字体（最小 18px，正文 24px+）
- 按钮：大圆角按钮，点击区域 >= 48x48px
- 图标：使用 emoji 或简单图标
- 动画：平滑过渡，避免快速闪烁

### 插画风格提示词模板
```
儿童绘本风格，温馨可爱，色彩明亮，卡通画风，
[场景描述]，适合3-8岁儿童观看，无恐怖元素
```

### 内容安全策略
- 预置中英文敏感词库（暴力、恐怖、成人内容等）
- 对故事内容进行扫描，标记风险等级
- 高风险内容拦截并提示家长审核
- 支持家长模式（输入密码后可编辑自定义内容）
